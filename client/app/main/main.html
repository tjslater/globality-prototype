This app was created with Node-Express-Mongo and Angular

It does the following:

Backend:

  • Creates a Mongoose Schema for products
  • Registers a pagination plugin with the parameters:
    • title
    • author
    • blurb
    • product_url
    • img_url
  • Creates an express App
  • Creates two API endpoints:
    • a getter, for pages of 20 items
    • a get single, by ID
  • Creates seed data for MongoDB
  • Sets environment variables for various production environments

Frontend

  • Creates this view
  • Creates a two-way bound catalog list view, /product
  • Organizes the catalog view responsively
  • Opens a modal on click of any item to show a larger picture, and more details
